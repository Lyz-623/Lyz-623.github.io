Yunze Li Academic Homepage
==========================

This repository contains an academic personal homepage for Yunze Li. It has been rebuilt from the previous course-blog content into a GitHub Pages-ready research profile.

Primary static entry:

```text
index.html
assets/main.css
assets/microscopy-hero.png
```

Hexo source is also kept for future maintenance:

```text
source/_data/profile.yml
themes/academic/
```

If Node/npm is available, the Hexo version can be generated with:

```bash
npm install
npm run build
```

For direct GitHub Pages deployment, serve the repository root or the `public` folder.
